W3b-wc-button

A Web Components Button

Make sure to add the following Pollyfills to your application:

<script src='https://unpkg.com/@webcomponents/webcomponentsjs@2.0.0/webcomponents-loader.js'></script> <script src='https://unpkg.com/@webcomponents/webcomponentsjs/custom-elements-es5-adapter.js'></script>

Text Buttons


example: <w3b-wc-button size="small" label="button" theme="secondary"></w3b-wc-button>

Icon Buttons



example: <w3b-wc-button size="small" label="" theme="secondary"> < span slot="icon-right"> <img src="./icon.png" /> </span> </w3b-wc-button>

Danger Button

example: <w3b-wc-button size="medium" label="Delete" theme="danger" disabled></w3b-wc-button>

CSS Variables

  • --w3b-fontsize-small:
  • --w3b-fontsize-medium:
  • --w3b-fontsize-large:
  • --w3b-opacity:
  • --w3b-icon-position:
  • --w3b-primary-color:
  • --w3b-primary-background-color:
  • --w3b-primary-border:
  • --w3b-secondary-color:
  • --w3b-secondary-background-color:
  • --w3b-secondary-border:
  • --w3b-danger-color:
  • --w3b-danger-background-color:
  • --w3b-danger-border:

Attributes

  • disabled: Boolean
  • label: String
  • theme: String - primary, secundary, danger
  • size: String - small, medium, large

Event

You can listen to the w3bbuttonclick event:

window.addEventListener('w3bbuttonclick', function (event) { console.log(event.detail); }, true);
Console result:
{isClicked: w3b-wc-button.danger} {isClicked: w3b-wc-button.create}
